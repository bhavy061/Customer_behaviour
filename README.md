# 📊 Customer Behaviour Analysis

## 📌 Project Overview

Customer Behaviour Analysis is a data analytics project focused on understanding customer purchasing patterns, preferences, and trends.

The objective of this project is to analyze customer data, identify meaningful insights, and understand factors influencing customer decisions. These insights can help businesses improve marketing strategies, customer engagement, and decision-making.

---

## 🎯 Objectives

- Analyze customer purchasing behaviour
- Identify customer trends and patterns
- Perform data cleaning and preprocessing
- Explore relationships between customer attributes
- Generate meaningful business insights through visualization

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
